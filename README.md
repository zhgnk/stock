（1）选股

不迷信基本面（企业财报）、不迷信消息面（内幕消息、新闻资讯概念、企业公告）、不迷信资金面（庄家/主力）、不迷信技术面（K线图）。
各种K线、技术指标（KDJ/MACD/TRIX）都是走出来的结果的可视化，不是推动力量，不要因果倒置。股票交易是独立交易事件，一把是一把。不要迷信。上一把和下一把没有半毛钱关系，但是现在的股票软件可视化都喜欢画成连贯的图形，这会误导人。

不风格选股。在中国这个市场，任何风格都不会长久都会经历一个被人追捧-捧到最高-逃散这样的周期。
把自己当做一个游泳者或冲浪者即可，顺水而为、逐水草而居，浪来就买、浪走就卖，潮来潮去什么都不为。

不抄底，不赚第一个铜板。中国股市永远没有底，永远要在个股上行周期中交易。永远不要担心自己错过一波。不要买在人声鼎沸处更不要买在无人问津处，但一定要卖在人声鼎沸处。

（2）下限-上限

止盈：5%-20%涨幅。不赚最后一个铜板。是炒股养家，要追求确定的、稳定可持续的盈利，而不是投机赌博，赌狗赌狗，赌到最后一无所有。

止损：5%-10%跌幅。错了就是错了，说明自己的交易系统还有问题，要总结反思，没有修补前不要乱动。
不对股票的涨跌进行归因，归因既无意义，还容易走入自己的信息茧房陷阱，还容易给自己虚妄-遮挡住以后致命的错误。
要出就出干净，不要存侥幸心理留一点再观察观察。不要给自己找借口。人是对抗不了人性的，你沉没越多你越不想放弃。常在河边走，必然有一天会湿鞋，想让鞋子不湿，只有远离河边，才能抗住自己的心魔。不要跌破5%还不出，因为人有沉没成本，越输越想等等幻想着翻盘，就生出侥幸赌的心理，赌狗赌狗，赌到最后一无所有。
一把一把来，不在一城一池的得失。股票交易是独立交易事件，一把是一把。不要迷信。上一把和下一把没有半毛钱关系。
但是，
庄家有周期：吸筹-震荡再吸足够筹码-拉升打榜吸引人-回调吸引人-庄家拉升出货正好散户入坑接盘。
信息扩散有周期：先知先觉-中知中觉-后知后觉-不明人。
人性有周期：人性扩散周期：贪婪-恐惧-贪婪往复循环。
所以要有庄家思维、老板思维、富人思维，你是谁的对手盘，谁是你的对手盘，要想清楚，要甘愿做上家的对手盘，要对自己的下家对手盘做提前设局。
不要有散户思维、员工思维、穷人思维。散户思维、员工思维、穷人思维是永远发不了财的。

（3）买入策略

先从利弗莫尔买入法开始。在盘口上行趋势背景下买，不要在盘口下行趋势背景下买。
但未来走向不加仓。一次性买足。不后悔买少了，也不后悔买多了。不断完善自己的交易系统。

（4）持有周期

持有周期：不超过5个交易日。中国这个市场，不是时间的朋友，十多年事实证明打板追龙这种方法有效。如果你的性格不适合打扮追龙，那么对不起，你可能不适合在A股炒股。
熟悉这池子水的水性，你就是浪里白条。不熟悉这池子水的水性，你就是淹死。
